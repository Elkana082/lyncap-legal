Privacy Policy for Lyncyap

Last Updated: February 2026

1. Information We Collect

  • Email address
  • Username
  • Encrypted password
  • GPS location data (with user permission)
  • Profile information
  • Posts including images, videos, and music
  • Comments, likes, follows, and interactions
  • Reports and block activity
  • Device and usage information

2. How We Use Your Information

  • To create and manage user accounts
  • To authenticate users using JWT (JSON Web Tokens)
  • To enable posting, messaging, commenting, and sharing
  • To allow blocking, reporting, and moderation
  • To improve performance and user experience
  • To send important account-related emails via Nodemailer

3. Location Data

Lyncyap may collect GPS location data to enhance user experience. Location data is collected only with user permission and is not sold or shared for advertising purposes.

4. Data Storage & Security

User account data is stored securely using MongoDB Atlas. Media files (images, videos, music) are stored using Cloudinary. We use JWT-based authentication and implement industry-standard security measures to protect user accounts.

5. Third-Party Services

  • MongoDB Atlas (database storage)
  • Cloudinary (media storage)
  • Nodemailer (email services)
  • Google Play Services

6. User Controls

Users may:

  • Delete posts, comments, or messages
  • Unlike or unfollow users
  • Block or unblock users
  • Report content or users
  • Request account deletion

7. Sharing to Other Apps

Users may share content to third-party apps. Lyncyap is not responsible for how external platforms use shared content.

8. Data Retention

We retain user data as long as the account remains active. Upon account deletion, personal data and associated media will be permanently removed, except where temporary retention is required for security or legal reasons.

9. Changes to This Policy

We may update this Privacy Policy periodically. Continued use of the app means acceptance of any updates.
